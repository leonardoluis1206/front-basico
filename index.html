Introdução ao CSS

CSS significa Cascading Style Sheets (Folhas de Estilo em Cascata).

Você pode aplicar CSS de três formas:

  • Inline (dentro da tag HTML)
  • Interno (dentro da tag <style>)
  • Externo (arquivo .css separado)
